Final Classes (since C++ 11)

A class or struct declaration marked as final cannot be derived from.

An example:

class A final {
  /* implementation */
};
class B : public A { // FAILS to compile!!!
  /* implementation */
};
An analogous behaviour is found in virtual functions that are marked as final.
