Mark-Down

Markdown is a lightweight markup language that you can use to add formatting elements to plaintext text documents. It was created by John Gruber in 2004.

Markdown syntax is designed to be readable and unobtrusive, so the text in Markdown files can be read even if it isn't rendered.

Basic Syntax

These are the elements outlined in John Gruber's original design document. All Markdown applications support these elements.

Element Markdown Systax
Heading 
# H1
  ## H2
  ### H3
Bold 
**bold text**
Italic 
*italicized text*
Blockquote 
> blockquote
Ordered List 
1. First item
  2. Second item
  3. Third item
Unordered List 
- First item
  - Second item
  - Third item
Code (single backticks) 
`code`
Horizontal Rule 
---
Link 
[title](https://www.example.com)
Image 
![alt text](image.jpg)
