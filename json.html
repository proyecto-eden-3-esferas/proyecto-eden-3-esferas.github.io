JSON is faster because it is designed specifically for data interchange. JSON encoding is terse, which requires fewer bytes for transit. JSON parsers are less complex, which requires less processing time and memory overhead. XML is slower, because it is designed for a lot more than just data interchange.

Is JSON more compact than XML?

JSON is a more compact format, meaning it weighs far less on the wire than the more verbose XML. JSON is easier to work with in some languages (such as JavaScript, python, and php). Formatted JSON is generally easier to read than formatted XML.

Is JSON compatible with JavaScript?

The JSON format is syntactically identical to the code for creating JavaScript objects. Because of this similarity, a JavaScript program can easily convert JSON data into native JavaScript objects.
